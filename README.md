# nodejs-web-vite-svelte-ssr-typescript-template

## Description
A svelte template for the vite packager.

Feature
- ssr server side rendering

## Tech stack
- npm
- vite
- svelte
- typescript

## Docker stack
- node

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
